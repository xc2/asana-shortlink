# Asana Shortlink Service

[![Deploy to Cloudflare](https://deploy.workers.cloudflare.com/button)](https://deploy.workers.cloudflare.com/?url=https://github.com/xc2/asana-shortlink)

This is a simple service enables direct linking to Asana tasks.

The link format is `https://<domain>/<custom_id>`, e.g., `https://asana.your-company.workers.dev/PROJ-123`.

This needs [ID Custom Fields](https://help.asana.com/s/article/id-custom-fields?language=en_US) to work.
